# Developer Agent System

Complete system for building AI coding assistants. Contains 5 production-ready CLAUDE.md configurations.

## Contents

1. **code-review-agent.md** - Automated code review with security focus
2. **documentation-agent.md** - Technical documentation generator
3. **testing-agent.md** - Test creation and coverage analysis
4. **refactoring-agent.md** - Code quality and refactoring suggestions
5. **security-audit-agent.md** - Security vulnerability detection

## Quick Start

1. Copy the relevant `.md` file to your project root
2. Rename to `CLAUDE.md` (for Claude Code) or `AGENTS.md` (for other tools)
3. Customize the configuration sections for your project
4. Start using with your AI coding assistant

## Usage

Each agent config is designed to work standalone or in combination. For best results:

- Start with one agent that matches your immediate need
- Customize the "Project Context" section with your tech stack
- Adjust safety rules based on your risk tolerance
- Add project-specific patterns to the relevant sections

## Customization Guide

### Project Context Section
Replace placeholders with your actual:
- Tech stack (languages, frameworks)
- Directory structure
- Naming conventions
- Build commands

### Safety Rules
Adjust based on your environment:
- Production vs development
- Team size and permissions
- CI/CD integration
