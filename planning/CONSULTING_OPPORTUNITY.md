## Claude Code Consulting Niche

### Why This Is Underserved

1. **Claude Code is new** - Launched Feb 2025, rapid evolution
2. **Steep learning curve** - Skills, hooks, subagents, prompting
3. **Best practices emerging** - No established playbooks yet
4. **Team adoption is hard** - Getting developers to use it effectively

### What Clients Need

| Service | Description | Rate Range |
|---------|-------------|------------|
| Setup & Config | Configure Claude Code for team, CLAUDE.md creation | $150-200/hr |
| Workflow Design | Design AI-assisted development processes | $200-300/hr |
| Team Training | Workshop on Claude Code best practices | $1,500-3,000/day |
| Prompt Engineering | Custom prompts/skills for specific workflows | $150-250/hr |
| Ongoing Advisory | Retainer for questions, updates, optimization | $2,000-5,000/mo |

### Target Clients

**Startups (Series A-C)**
- Moving fast, want latest tools
- Small teams, can adopt quickly
- Budget: $5-20K for tooling/training

**Mid-size Tech Companies**
- Rolling out to 20-100 developers
- Need standardized approach
- Budget: $20-50K for rollout

**Agencies/Consultancies**
- Using AI for client work
- Need efficiency gains
- Budget: $10-30K for training

**Enterprise (Exploratory)**
- Pilots in specific teams
- Compliance/security concerns
- Budget: Variable, procurement heavy

## The Validation Question

Before investing heavily in consulting, validate:

1. **Would someone pay you $150/hr for Claude Code advice?**
   - Test: Offer 1-hour paid consultation
   - If yes: Continue
   - If no: Adjust offer or abandon

2. **Can you consistently find leads?**
   - Test: 4 weeks of community presence
   - If yes: Systematize
   - If no: Try different channels

3. **Is this sustainable with other commitments?**
   - Test: 5-10 hours/week capacity
   - If yes: Scale carefully
   - If no: Keep smaller
